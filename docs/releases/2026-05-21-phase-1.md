# Release Notes — 21 May 2026

## Nieuwe Features

- [IA-CHIP] Chip click toggles realtime filter outside edit mode (label, shop, assignee, date)
- [IA-SUBTASK] Subtask attribute with inline expandable UI + add subtask button
- [GROC-SWAP] Grocery items limited to shop/swap/delete attributes (removed irrelevant task attributes)
- [PAPERLESS-001] Paperless-ngx 'todoless' tag → Task with subtasks (Controleren/Verwerken/Archiveren)
- [UX-CHIPS] Labels and shops standardized as rounded chips across all views
- [UX-SEARCH] Consistent searchbar placeholders: "Search tasks", "Search inbox", "Search groceries"

## Verbeteringen

- [SET-TOKEN] API token creation simplified to single-field form (name only, permissions/expiry auto-default)
- [SET-PROF] Settings profile consolidated into single "Your Profile" block (removed duplicate email/role blocks)
- [UX-MEMBERS] Member status badges made consistently sized (text-xs)
- [DOC-SWAGGER] OpenAPI spec completed with response schemas, examples, and 401 error responses for all ~120 endpoints

## Bug Fixes

- [SET-PERSIST] Profile edit persistence fixed (snake_case vs camelCase mismatch: displayName → display_name)
- [MULTI-SHARED] Family members can now edit each other's non-private tasks and items (update/delete rules)
- [IA-EDIT] Edit mode restricted to hamburger menu only with 60s inactivity timeout + visual indicator
